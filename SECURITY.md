# Security Policy

## Supported Versions

This repository is documentation-only and does not ship executable software.
Security concerns are still welcome, especially for potentially harmful,
unsafe, or misleading guidance.

## Reporting a Vulnerability

If you identify a security issue in the repository content, please open an issue:
https://github.com/suayip-isik/claude-golden-rules/issues

When reporting, include:

- A clear description of the issue
- Where it appears (file + section)
- Potential impact
- Suggested correction (if available)

Reports are reviewed as quickly as possible.
