# Contributing

Thank you for helping improve this guide. Contributions of all kinds are welcome — corrections, new sections, clarifications, and Turkish translations.

## Ways to Contribute

- **Report an error** — open an issue if something is factually wrong or outdated
- **Suggest an addition** — open an issue describing the gap you want filled
- **Submit a correction** — open a pull request with the fix

## Pull Request Process

1. Fork the repository
2. Create a branch: `git checkout -b fix/description` or `feat/description`
3. Make your changes
4. Open a pull request with a clear description of what changed and why

## Style Guidelines

- Keep sentences concise and direct — match the tone of the existing guide
- Prefer concrete examples over abstract explanations
- Avoid redundancy with existing sections; link instead
- Tables and code blocks are preferred over long prose for reference material

## Language Policy

- **English** is the primary language for all guide content
- **Turkish** translations are maintained alongside English versions
- If you update content in English, please update the Turkish counterpart if you are able to — or note in your PR that a Turkish update is needed

## What Not to Submit

- General Claude.ai or Anthropic API documentation (this guide covers Claude Code CLI only)
- Promotional content or links to commercial products
- Content that contradicts Anthropic's official documentation without a clear rationale

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.
